# tears-letf-strategy

## Setup

- install dependencies
- add a `.env` file with a FRED API key

## Scripts

Run the application using `tsx`

```sh
$ yarn start
```

Start the development server with file watching

```sh
$ yarn dev
```

Run tests using Jest

```sh
$ yarn test
```
